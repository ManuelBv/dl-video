dl-video

Detect and download video assets from any web page.

⬇ Download Extension (.zip)

How to install

  1. Download the .zip file above and unzip it.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode (top-right toggle).
  4. Click Load unpacked and select the unzipped dist/ folder.
  5. The dl-video icon will appear in your toolbar. Click it to open the side panel.
